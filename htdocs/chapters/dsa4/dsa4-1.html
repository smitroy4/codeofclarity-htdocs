Lets Start!

Array is a collection of items of the same variable type that are stored at contiguous memory locations. It is one of the most fundamental and widely used data structures in programming due to its simplicity and efficiency.

Basic Terminologies of Array

  • Array Index: The position of elements within the array, starting from 0.
  • Array Element: The actual data stored at each index.
  • Array Length: The number of elements that the array can hold.

Memory Representation of Array

All elements in an array are stored in contiguous memory blocks. This memory layout allows constant-time access using the index, which makes operations like traversal and element access efficient.

Declaration of Arrays

// Integer array
int arr[5];

// Character array
char arr[10];

// Float array
float arr[20];

Initialization of Arrays

int arr[] = {1, 2, 3, 4, 5};
char arr[5] = {'a', 'b', 'c', 'd', 'e'};
float arr[10] = {1.4, 2.0, 24, 5.0, 0.0};

Why Do We Need Arrays?

Arrays help in storing multiple items of the same type efficiently. Instead of creating separate variables for each item (like marks of many students), we use arrays to hold and manage all items together, which makes it easier to manipulate and maintain data.

Types of Arrays

Based on Size

1. Fixed Sized Arrays

Fixed-size arrays have a predefined size and cannot be resized during runtime. Memory is allocated statically or dynamically but remains unchanged.

// Compile-time allocation
int arr[5];
int arr2[5] = {1, 2, 3, 4, 5};

// Dynamic allocation
int* arr = new int[5];

2. Dynamic Sized Arrays

Dynamic arrays grow or shrink in size during runtime as needed. Memory is allocated and deallocated dynamically.

#include <vector>
vector<int> v;

Based on Dimensions

1. One-dimensional Array

A 1D array stores elements in a linear sequence like a single row.

2. Multi-dimensional Array

Multi-dimensional arrays can represent more complex data structures like matrices or tables.

  • 2D Arrays: Represent rows and columns (matrices).
  • 3D Arrays: Arrays of 2D arrays, useful in 3D modeling and scientific computations.

Operations on Arrays

1. Array Traversal

Traversal means accessing each element of an array in a sequence.

Types:

  • Sequential Traversal: From index 0 to the last index.
  • Reverse Traversal: From the last index to 0.

2. Insertion in Array

Inserting an element involves shifting existing elements if not inserting at the end.

Types:

  • At Beginning: Shift all elements right.
  • At Specific Index: Shift elements after the index.
  • At End: Append (efficient for dynamic arrays).

3. Deletion in Array

Deleting an element involves shifting the subsequent elements left.

Types:

  • At Beginning: Shift all elements left.
  • At Specific Index: Shift only elements after the index.
  • At End: Reduce size (in dynamic arrays).

4. Searching in Array

Searching is used to locate a specific element in the array.

Types:

  • Linear Search: Sequentially compares each element.
  • Binary Search: Efficient search for sorted arrays using divide-and-conquer.
